## css 选择器的优先级别

1. 选择器的针对性越强, 它的优先级别越高

2. 选择器的权重值

   + 
   + important: 1000
   + id 选择器: 100
   + 类选择器: 10
   + 伪类, 伪元素: 10
   + 标签选择器: 1
   + 通用选择器 (*) : 0

3. 原则

   选择器的权重值累加在一起, 大的优先, 如果权重值相等, 后面定义的会覆盖前面的定义

4. 优先级排序

   | 选择器                      | 权重值 | 级别排序 |
   | --------------------------- | ------ | -------- |
   | important                   | 1000   | 1        |
   | 内联 (在标签内容指定 style) |        | 2        |
   | id                          | 100    | 3        |
   | 类, 伪类, 伪元素            | 10     | 4        |
   | 标签                        | 1      | 5        |
   | 通用 (*)                    | 0      | 6        |
   | 继承                        |        | 7        |
